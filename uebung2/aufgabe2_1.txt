a) Die Variable xi in "LOOP xI DO P END" gibt an wie oft das Programm P ausgeführt werden soll. 
Der Counter, der jeden Programmdurchlauf um eins erhöht wird, steigt und erreicht somit irgendwann den Wert xI, da xi 
nicht unendlich ist. Im Programm selbst kann auch nur eine endliche Menge an weiteren Programmaufrufen geschehen, die alle 
selbst terminieren, da ebenfalls LOOP-Programme. 

b)
x1 = 1;
WHILE x1 != 0 
DO 
x1 := x1 + 1; 
x1 := x1 - 1 
END 

c)
LOOP x1 DO P END

wird umgeformt zu 

WHILE x1 != 0 
DO 
P
x1 := x1 - 1
END 

d)
i)      x + y

P1;
WHILE y != 0
DO
x := x + 1
y := y - 1
END


ii)     x * y
x1 = y
x2 = x
x3 = 0
WHILE x1 != 0
DO
    WHILE x2 != 0 
    DO
        x3 := x3 + 1
        x2 := x2 - 1 
    END
    x1 := x1 - 1
END
x3 enthält das Ergebnis


iii)    x ^ y
x1 = x
x2 = y
x3 = 1
x4 = x2

WHILE x2 != 0
DO
    WHILE x1 != 0
    DO
        WHILE x4 != 0 
        DO
            x3 := x3 + 1
            x4 := x4 - 1 
        END
        x4 = x3
        x1 := x1 - 1
    END
    x1 := x1 - 1
END

x3 enthält das Ergebnis der Berechnung
